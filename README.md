# Калькулятор учета продаж (Python)

Простое и надежное консольное приложение для малого бизнеса.

### Что умеет программа:
* Считает выручку по каждому товару и общую сумму за день.
* Формирует список проданных позиций (чек).
* **Защищена от ошибок**: если пользователь введет буквы вместо цифр, программа не «упадет».
* Автоматически сохраняет итоги дня в текстовый файл `report.txt`.

### Как запустить:
Нужен Python 3. Просто запустите файл: `python stort.py`
